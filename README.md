# web3-learning-notes
Web3 & 区块链学习 自学Solidity、Python链上工具开发

## 项目目录
- `balance-query-tool/`：0G链/以太坊余额查询小工具（Python）
- `solidity-practice/`：Solidity智能合约练习（ERC20代币、基础合约）
- `learning-notes/`：学习笔记、技术总结

## 完成内容
- ✅ 0G链余额查询工具（Python）
- ✅ 简单ERC20代币合约（Solidity）

## 运行环境
- Python 3.8+
- Solidity ^0.8.0
- 需要安装requests库：`pip install requests`
